True and False

In predicate logic, the propositional constants ⊤ (true) and ⊥ (false) are governed by specific rules:

True Introduction (⊤ intro):

  • The true introduction rule produces a proof of ⊤ (true) in any context, indicating that "true" can be derived universally.

False Elimination (⊥ elim):

  • From a proof of ⊥ (false), false elimination gives a proof of any proposition α, denoted as exfalso, emphasizing that anything follows from falsehood.

The negation elimination theorem illustrates the application of false elimination, asserting that if there are proofs of both p and ¬p, any proposition q can be derived.
