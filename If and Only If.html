If and Only If

The "If and Only If" rule in predicate logic consists of elimination and introduction:

If and Only If Elimination:

  • Left If and Only If Elimination (↔ elimL): From a proof of α ↔ β, this produces a proof of α → β.

  • Right If and Only If Elimination (↔ elimR): From a proof of α ↔ β, this produces a proof of β → α.

If and Only If Introduction:

  • From proofs of α → β and β → α, If and Only If Introduction (↔ intro) gives a proof of α ↔ β.

This rule allows deducing both directions of an "if and only if" statement from its individual implications. The text also mentions the symmetry of conjunction ( ∧ ), expressing that (a ∧ b) ↔ (b ∧ a).

If and Only If Elimination:

Example: Prove p on the assumptions h₁ : p ↔ q and h₂ : q.

In Lean: (h₁ : p ↔ q) (h₂ : q) : p :=

Proof: We have h3 : q → p by right iff elimination on h1. From h3, it suffices to show q, which follows from h2.

In Lean:

begin

have h₃ : q → p, from iff.elim_right h₁,

apply h₃,

show q, from h₂,

end

    IFF Elimination: on h. This gives two implications: h1 : p → (q ∧ r) and h2 : (q ∧ r) → p.

    Implication Elimination: on k and h1. This implies (q ∧ r).

    And Elimination: to (q ∧ r). This step allows us to conclude q from the given assumptions.

If and Only If Introduction:

Example: Given h : p ↔ q, derive q ↔ p.

In Lean: lemma (h : p ↔ q) : q ↔ p :=

Proof: We have h2 : p → q and h3 : q → p by ↔ elimL and ↔ elimR on h. The result follows by ↔intro on h3 and h2.

In Lean:

begin

have h₂ : q → p, from iff.elim_left h₁,

have h₃ : p → q, from iff.elim_right h₁,

show q ↔ p from iff.intro h₁ h₂,

end
