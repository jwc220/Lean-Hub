Exam Exercises

Propositional Logic

Section A Questions:

This question is taken from the January 2022 examination:

In each part of this question, select one or more possible deductions from a list of premises. You need not provide reasoning.
  1. Given the premises P and P ∨ Q, which of the following can be deduced : P, Q, P ∧ Q, P ∨ Q, ¬P?
  2. Given the premises P and P → (P → Q), which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
  3. Given the premises Q and Q → (P → Q), which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
  4. Given the premise ¬P → Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?

This question is taken from the January 2023 examination:

In each part of this question, select one or more possible deductions from a list of premises. You need not provide reasoning.
  1. Given the premises p ∧ q and p, which of the following can be deduced: p, q, p ∧ q, p ∨ q, ¬p, ¬q?
  2. Given the premises p ∨ q and q, which of the following can be deduced: p, q, p ∧ q, p ∨ q, ¬p, ¬q?
  3. Given the premises p → q and q, which of the following can be deduced: p, q, p ∧ q, p ∨ q, ¬p, ¬q?
  4. Given the premises p → q and ¬q, which of the following can be deduced: p, q, p ∧ q, p ∨ q, ¬p, ¬q?
(8 marks)

This question is taken from the January 2022 examination:

In each part of this question, select one or more possible deductions from a list of premises. You need not provide reasoning.
  1. Given the premises P and ¬Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
  2. Given the premises ¬P and Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
  3. Given the premise P ∨ Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
  4. Given the premises P → Q and P, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ¬P?
(8 marks)

This question is taken from the January 2021 examination:

In each part of this question, select one or more possible deductions from a list of premises. You need not provide reasoning.
  1. Given the premises P → Q and P, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ⊥?
  2. Given the premises P → Q and Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ⊥?
  3. Given the premises P ∨¬P and Q∨¬Q, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ⊥?
  4. Given the premise P ∧ ¬P, which of the following can be deduced: P, Q, P ∧ Q, P ∨ Q, ⊥?
(8 marks)

Section B Questions:

This question is taken from the January 2020 examination:

Using rules of inference for the propositional calculus,
  1. deduce P ∧ (Q ∨ R) from the premises P and Q;
  2. deduce S from the premise (S ∨ T) ∧ (T =⇒ S).
You need not explicitly name each rule of inference.

(10 marks)

This question is taken from the January 2021 examination:

Let P, Q, and R be propositions. Given h1 : P ↔ Q ∨ R, h2 : P, and h3 : ¬Q, prove R. Your proof should only use the rules of inference of the propositional logic and not depend on classical reasoning.

(12 marks)

This question is taken from the January 2022 examination:

Let A and B be propositions. Prove that (B ∨ ¬A) → (A → B) using only the introduction and elimination rules of logic.

(12 marks)

This question is taken from the January 2023 examination:

Give a Fitch-style proof of ((p ↔ q) ∧ ¬q) → ¬q, for propositions p and q. Your proof should use only the rules of inference.

(12 marks)
