Propositional Logic

Propositional logic deals with statements that can be true or false. In mathematics, it is essential for studying logical relationships between propositions.

Propositional logic introduces a formal language for deduction, where theorems take a general symbolic form. We will go through 8 different rules of logic to support you in your studies.

Rule Symbol Lean Input Connection Meaning
And ∧ \and P ∧ Q (P ∧ Q) is true if both P and Q are true otherwise false.
Implication → \to, \r, \imp P → Q If P happens then Q happens.
Or ∨ \or P ∨ Q (P ∨ Q) is true if either P or Q is true (or both) otherwise false.
If and Only If ↔ \iff, \lr P ↔ Q P happens if and only if Q happen.
True ⊤ true p → ⊤ P is always true.
False ⊥ false p → ⊥ P is always false.

Notes and Exercises for the following rules

1. And

The "And" rule in predicate logic allows you to deduce components of a conjunction.

2. Implication

In logic, implication is relationship between different propositions where the second proposition is a logical consequence of the first.

3. Or

The "Or" rule in predicate logic comprises Or Introduction, allowing the derivation of a disjunction from proofs of its components, and Or Elimination, enabling the deduction of a proposition from a disjunction and separate proofs under each disjunct assumption.

4. If and Only If

The "Or" rule in predicate logic comprises Or Introduction, allowing the derivation of a disjunction from proofs of its components, and Or Elimination, enabling the deduction of a proposition from a disjunction and separate proofs under each disjunct assumption.

5. True and False

In predicate logic, the rules include true introduction (⊤ intro) allowing universal derivation of true, false elimination (⊥ elim) enabling the derivation of any proposition from falsehood, and the negation elimination theorem asserting that proofs of both a proposition and its negation permit the derivation of any other proposition.
